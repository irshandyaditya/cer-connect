# cer-backend

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.2.20. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.

Langkah jalanin:
bun install
cp .env.example .env
bun run prisma:gen
bun run prisma:migrate
bun run dev
